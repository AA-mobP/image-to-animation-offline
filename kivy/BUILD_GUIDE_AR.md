# بناء نسخة CLI للويندوز

## نظرة عامة
يمكنك بناء نسخة CLI مستقلة بدون الواجهة الرسومية، مما يجعل الملف أصغر حجماً وأسرع في التشغيل.

## المتطلبات
```bash
pip install pyinstaller
```

## طريقة البناء

### 1. بناء CLI فقط (موصى به للاستخدام من سطر الأوامر)
```bash
cd kivy
pyinstaller cli.spec --clean
```

**المخرجات**:
- الملف: `dist/Img2Sketch-CLI.exe`
- الحجم: ~50-80 MB (أصغر بكثير من النسخة الكاملة)
- الاستخدام: من CMD/PowerShell فقط

**مثال على الاستخدام**:
```bash
.\dist\Img2Sketch-CLI.exe -i photo.jpg -o ./output -slr 10-40 -fr 30 -dc
```

### 2. بناء الواجهة الرسومية (GUI)
```bash
cd kivy
pyinstaller dlDesktop.spec --clean
```

**المخرجات**:
- الملف: `dist/Img2Sketch.exe`
- الحجم: ~150-200 MB
- الاستخدام: واجهة رسومية كاملة

## الفروقات بين النسختين

| الميزة | CLI | GUI |
|--------|-----|-----|
| الحجم | ~50-80 MB | ~150-200 MB |
| الاستخدام | سطر الأوامر فقط | واجهة رسومية |
| السرعة | أسرع في التشغيل | أبطأ قليلاً |
| المعاملات | يدعم جميع المعاملات | تحكم عبر الواجهة |
| الأتمتة | مثالي للسكريبتات | غير مناسب |

## بناء النسختين معاً

إذا أردت بناء النسختين في نفس الوقت:

```bash
cd kivy

# بناء CLI
pyinstaller cli.spec --clean

# بناء GUI
pyinstaller dlDesktop.spec --clean
```

ستجد:
- `dist/Img2Sketch-CLI.exe` - نسخة CLI
- `dist/Img2Sketch.exe` - نسخة GUI

## استكشاف الأخطاء

### المشكلة: "Missing dependencies"
```bash
pip install -r requirements.txt
```

### المشكلة: الملف كبير جداً
- استخدم `cli.spec` بدلاً من `dlDesktop.spec`
- CLI لا يحتوي على مكتبات الواجهة الرسومية

### المشكلة: "DLL load failed"
- تأكد من تثبيت Visual C++ Redistributable
- أعد تشغيل الكمبيوتر بعد التثبيت
